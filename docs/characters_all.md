# ZeroDayMMO Character Documentation

## Nova — HeXtress

**Role:** Offensive Debuffer/Caster
**Abilities:** Overloads enemies with hex chains, voltage surges, and debuffs (slow, weaken defenses).
**Solo Play:** High burst damage with short self-sustain. Focuses on chaos and quick kills.
**Personality:** Bold, reckless, thrives on disruption.

\n::: Ability Tree \n- **Passive:** Overclocked Circuits (Shock stacks, stun/slow)\n- **Abilities:** Hex Surge, System Fracture, Static Snare\n- **Ultimate:** Blackout Cascade\n\n::: Progression
- Level 1: Hex Surge, System Fracture\n- Level 3: Static Snare\n- Level 10: Blackout Cascade (ultimate)\n
